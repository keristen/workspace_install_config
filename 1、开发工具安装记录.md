## 1、Anaconda 安装
```
wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/Anaconda3-2024.02-1-Linux-x86_64.sh

bash Anaconda3-2024.02-1-Linux-x86_64.sh
```

## 2、cuda 11.8安装
```
wget https://developer.download.nvidia.com/compute/cuda/11.8.0/local_installers/cuda_11.8.0_520.61.05_linux.run

sudo sh cuda_11.8.0_520.61.05_linux.run
```
检查版本
```
nvcc -V
```
## 3、cudnn8.9.7 for cuda 11 安装
```
tar -xvf  cudnn-linux-x86_64-8.9.7.29_cuda11-archive.tar.xz
```

将文件夹名称改为 cuda
```
sudo cp cuda/include/cudnn*.h /usr/local/cuda-11.8/include

sudo cp cuda/lib/libcudnn* /usr/local/cuda-11.8/lib64

sudo chmod a+r /usr/local/cuda-11.8/include/cudnn*.h /usr/local/cuda-11.8/lib64/libcudnn*
```
检测cudnn版本，旧版cudnn需要将文件名改成cudnn.h
```
cat /usr/local/cuda/include/cudnn_version.h | grep CUDNN_MAJOR -A 2
```

## 4、Isaac gym安装：

下载安装包：  https://developer.nvidia.com/isaac-gym/download
```
tar -zxvf IsaacGym_Preview_4_Package.tar.gz

conda create -n rlgpu python=3.8 
conda activate rlgpu
cd isaacgym/python
pip install -e . --use-pep517
```
